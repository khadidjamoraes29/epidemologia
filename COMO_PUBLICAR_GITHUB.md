# Como publicar no GitHub

1. Entre em sua conta no GitHub.
2. Clique em **New repository**.
3. Use um nome como `leptospirose-seir-automato`.
4. Selecione **Public**, caso o professor precise acessar sem autorização.
5. Não marque a criação automática de README, pois o projeto já possui um.
6. Crie o repositório.
7. Na página seguinte, escolha **uploading an existing file**.
8. Envie todos os arquivos e pastas deste projeto.
9. Clique em **Commit changes**.
10. Teste o endereço em uma janela anônima.

O artigo deve apontar para o endereço geral do repositório e para:

```text
prompts/prompt_used.md
```

Antes da entrega, substitua no artigo os marcadores de link pelo endereço real do GitHub.
